# Redes neuronales para clasificar imágenes de gatitos y perritos

El objetivo de este proyecto es crear una red neuronal que identifique perros o gatos en una imagen. 

Este repositorio contiene los notebooks utilizados para el desarrollo del proyecto final realizado para la materia de redes neuronales y aprendizaje profundo de quinto semestre. 

Se incluyen tres notebooks, el primero, para el EDA contiene el análisis exploratorio y los gráficos creados. 

Los archivos de baseline y modelo final contienen el código para los modelos correspondientes. Al correrlos tendrás una carpeta zip con todos los modelos de las épocas y datos importantes de las métricas del último modelo. Puedes cambiar el modelo a evaluar siguiendo los comentarios. 

El archivo app.py tiene instrucciones para correr una app web muy sencilla que permite probar el modelo que gustes siempre y cuando haya sido creado con el script del modelo final.
